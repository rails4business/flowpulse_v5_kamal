YAML → JSON Tree Builder

Scrivi l’albero in YAML, copia il JSON

Il YAML è più leggibile per costruire percorsi annidati. Quando hai finito, converti e incolla il JSON nello stepper.

Struttura consigliata

Nodo normale

Usa id, title, description, question, color e children.

Nodo finale

Quando non ha children, può avere service_id, href e cta_label.

Icone e colori

Puoi usare icon_url e colori Tailwind: sky, red, emerald, amber, violet, blue, orange...

Campi disponibili

id — slug tecnico, es. salute

title — titolo visibile

description — testo breve

question — domanda dello step attivo

color — colore UI

icon_url — URL icona/logo quadrato

type — service, course, experience, event...

brand — PosturaCorretta, Flowpulse...

service_id — ID finale del servizio

href — link finale

cta_label — testo del bottone

children — figli del nodo
